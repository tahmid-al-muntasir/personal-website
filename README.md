## 📌 Overview
The source code for my personal digital architecture. This platform serves as a centralized hub for my engineering projects, technical write-ups, research preprints, and open-source contributions. 

Built with performance, SEO, and academic aesthetics in mind, it bridges the gap between textile engineering and artificial intelligence.

## 🛠️ Stack & Architecture
- **Frontend:** Next.js (App Router), React, Tailwind CSS
- **Backend & Content:** Sanity CMS
- **Deployment:** Vercel

## ⚙️ Local Development
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables (Sanity Project ID, Dataset).
4. Run the development server: `npm run dev`
5. Access the studio: `npm run sanity dev`
